Inference for the binomial proportion, likelihood, a priori distributions

Binomial distribution

Previously was presented the binomial distribution. The probability function corresponds then to:

$$ Y \sim \text{Bin}(n, \pi) \equiv f(y \mid n, \pi) = {n \choose y} \, \pi^{y} \ (1 - \pi)^{n-y} $$

An exmaple for example is the number of male children in a family. If the family has three children, where probability of being male is a half, then \(n = 3\) and \(\pi = 0.5\). If we compute the probability that \(P(Y = y)\), for example, the probability that the family has, for instance \(y=1\) male, is computed with the binomial distribution.

Some characteristics of this distribution are:

  • The mean is equal to \(n\pi\). For the aforementioned example is \(3 * 0.5 = 1.5\).
  • The variance corresponds to \(n\pi(1-\pi)\). In this example we obtain \(3 * 0.5 (1- 0.5) = 0.75\).
  • The random variable Y is discrete, taking the values \(y=0, 1, 2 ... , n\). The number of children can be 0, 1, 2 or 3.
  • The distribution is symmetric.
  • We can approximate the binomial distribution to a normal distribution for large values of \(n\). For instance \(n>25\) for \(\pi = 0.5\).
  • The probability function, the probability \(\pi\) is fixed, and what varies is only the random \(y\) value.

Likelihood

If instead of fixing the probability \(\pi\) we fix the \(y\) value, then we get the likelihood, where the random variable is the probability.

$$ Y \sim \text{Bin}(\pi, n) \equiv l(\pi \mid n, y) = {n \choose y} \, \pi^{y} \ (1 - \pi)^{n-y} $$

Since \(n\) and \(y\) are fixed, what varies is the part after the choose, called the likelihood kernel. The Likelihood is proportional to the kernel.

$$ l(\pi \mid n, y) = {n \choose y} \ \pi^{y} \ (1 - \pi)^{n-y} \propto \pi^{y} \ (1 - \pi)^{n-y} $$

Assignment 4

McBride et al. (2002) report the result of a research program on human health risk from contact with contaminated water. The researchers found that, out of \(n=116\) one-liter water samples from sites with heavy environmental impact, \(y=17\) samples contained Giardia cysts.

Delineate the probability function of \(Y\) (number of samples containing G. cysts) assuming \(17/116\) is the true probability.

Plot the likelihood function. What is its domain?

Solution

The binomial distribution follows

$$ \text{Bin}(n=116, \pi=17/116=0.147) $$

We first compute the probability function \(P(Y=0)\), where the random variable is \(y\). The R code to compute the probability function.

# Constants
abscissa <- c(0:116)
n <- 116
y <- 0
p <- 17/116

# Probability function for P(Y=0):
(choose(n, y)) * (p**y) * ((1-p)**(n-y))
# 1.038857e-08.

# Probability function for P(Y=10):
y <- 10
(choose(n, y)) * (p**y) * ((1-p)**(n-y))
# 0.01889021

# We define a probability function
bin_prob <- function(y, n, p) {
    (choose(n, y)) * (p**y) * ((1-p)**(n-y))
}

# We obtain all the probabilities applied to all the values of abscissa
ordenate <- bin_prob(y=abscissa, n=116, p=17/116)

# We plot the values of ordenate to visualize the probability distribution
plot(abscissa, ordenate, type="h"
   , xlab="Número de muestras contaminadas y"
   , ylab="P(Y = y)"
   , xlim=c(0,40))

The second part of the exercise asks to plot the likelihood function, and define its domain. The domain of \(\pi(0, 1)\), that is, it varies between \(0\) and \(1\).

The R code of the likelihood is trivial.

# Constants
abscissa_l <- c(0:1000)/1000

# We will use bin_prob, now assigning variable values to the probability
ordenate_l <- bin_prob(y=17, n=116, p=abscissa_l)

# Then we plot
plot(abscissa_l, ordenate_l, type="l"
   , xlab=expression(pi)
   , ylab="Likelihood"
   , xlim=c(0,0.4))

Election of the prior distribution

The posterior distribution is proportional to the likelihood times the prior, which we assign a constant \(K\). $$ \begin{align} \text{posterior}(\pi \mid n, y) &\propto l(\pi \mid n, y) \times \text{prior}(\pi) \\\\ &\propto \pi^{y} \ (1-\pi)^{n-y} \times K \\\\ &\propto \pi^{y} \ (1-\pi)^{n-y} \end{align} $$ The distribution does not add up to one, so we need to normalize in order to obtain a cumulative probability of one. The posterior distribution is: $$ (\pi \mid n, y) \sim \text{Be}\,(y+1, n-y+1) $$

The beta distribution

The beta distribution is continuous. $$ \pi \sim \text{Be}\,(a,b) \equiv f(\pi) = \frac{\Gamma(a+b)}{\Gamma(a) + \Gamma(b)}\,\pi^{a-1}(1-\pi)^{b-1} $$ With \(a>0\) and \(b>0\). The gamma function equals: $$ \Gamma(a) = \int_{0}^{\infin}x^{a-q}e^{-x}dx = (a-1)\,\Gamma(a-1) $$ Some characteristics of the beta distribution:

  • The mean is \(m=a/(a+b)\).
  • The variance is \(m(1-m)/(a+b+1)\).
  • The distribution is symmetric if \(a=b\), and asymmetric otherwise.
  • \(\text{Be}(1,1)\) is the uniform distribution.
  • The distribution is approximately normal for large \(a\) and \(b\).
